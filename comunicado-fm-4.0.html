Recado importante para você que utiliza o Fiscal manager A partir da versão 4.0 do layout da NFC-e será definido unicamente o protocolo TLS (Transport Layer Security – Segurança da Camada de Transporte) 1.2 ou versão superior como padrão de comunicação, segundo a NT 2016.002 (página 9). Ou seja, não será mais permitida à comunicação via protocolo SSL (Secure Sockets Layer – Protocolo de Camada de Segurança). O motivo desta mudança é a falta de segurança comprovada no uso desse Protocolo. Qual os sistemas operacionais afetados? • Windows XP; • Windows Vista; • Windows Server 2003. Quais sistemas poderão ser utilizados? • Windows 7 ou superior. Atentem-se à incompatibilidade das versões anteriores do Windows 7 com o layout 4.0 da NFC-e e evite problemas! Lembrando que a obrigatoriedade do layout 4.0 será a partir de 01 de outubro de 2018.
